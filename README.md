# Amruta Patil Portfolio

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript**. The portfolio showcases skills, projects, experience, certifications, and contact information with a clean UI and interactive animations.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Modern dark-themed user interface
* Animated hero section
* Custom cursor effects
* Smooth scrolling navigation
* Scroll reveal animations
* Animated statistics counters
* Mobile-friendly navigation menu
* Skills showcase section
* Projects section
* Experience and certifications timeline
* Contact section with social links

## Technologies Used

* HTML5
* CSS3

## Installation & Setup

### Method 1: Open Directly

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html`.

### Method 2: Run Using VS Code Live Server (Recommended)

1. Install Visual Studio Code.
2. Install the Live Server extension.
3. Open the project folder in VS Code.
4. Right-click `index.html`.
5. Select **Open with Live Server**.

The website will open in your browser automatically.
